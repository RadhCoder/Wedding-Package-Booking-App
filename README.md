## Project Overview
The **Wedding Package Booking App** is a mobile application that makes exploring and booking wedding packages simple and efficient. It offers users a single platform to compare Malay, Chinese, and Indian wedding packages, view pricing, guest capacities, and details of included services.

The app aims to streamline the booking experience with an intuitive, organized interface.

- **Detailed Information** ‚Äì Access guest limits, pricing, and service breakdowns.  
- **Smooth Booking Workflow** ‚Äì Navigate and book packages with minimal effort.  
- **User-Friendly Design** ‚Äì Clear, intuitive screens for an improved user experience.

## Challenges Addressed
Before this app, users struggled to find a **centralized digital platform** for wedding package selection. Information was scattered, comparing packages was tedious, and booking lacked convenience.

## How the App Solves It
- Centralizes wedding package information in one mobile platform  
- Provides clear comparisons of pricing, capacity, and services  
- Simplifies decision-making and booking processes through a smooth interface
